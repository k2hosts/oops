// Under the hood
[01]
Privacy Scrubber
A custom redaction engine runs inside the Cloudflare Worker. If the API crashes, error logs could expose your message content or API key. We replace them with [REDACTED_PRIVACY] before they leave the server.
[02]
Discord Health Monitor
Every 429 rate-limit and 500 server error fires an instant Discord webhook alert. Fix incidents before users report them — not after.
[03]
Smart Context Truncation
Large document uploads are auto-measured and trimmed to stay under the 6,000 TPM limit. The middle is cut, not the edges — so the model gets the beginning and end of your file intact.
[04]
Zero-Retention Identity Layer
Your Discord UID lives in Cloudflare KV for rate-limiting. Your conversation text lives nowhere. These two datastores have zero connection to each other — who you are is fully isolated from what you said.
[05]
Unopinionated Edge Routing
Headers are configured pass-through. No hidden system prompt. No "You are a helpful assistant." No AI moralizing pre-processing. If you send a question, the model answers it. Full stop.
